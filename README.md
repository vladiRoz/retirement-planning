# Retirement Planning Website

A comprehensive retirement planning website with four financial calculators to help users plan their financial future.

## Features

- **Responsive Design**: Mobile-friendly interface built with Next.js and Tailwind CSS
- **Four Financial Calculators**:
  - **Retirement Calculator**: Project retirement savings based on current age, savings, and contributions
  - **Savings Calculator**: Calculate compound interest growth with various compounding frequencies
  - **Investment Calculator**: Visualize portfolio allocation and expected returns based on risk profile
  - **Tax Calculator**: Estimate tax liability and take-home pay based on income and deductions
- **Interactive UI**: Real-time calculations with sliders, input fields, and dropdowns
- **Data Visualization**: Charts and graphs to visualize financial projections
- **Export Functionality**: Export results to CSV for further analysis

## Technologies Used

- **Frontend**: Next.js with TypeScript
- **Styling**: Tailwind CSS
- **Charts**: Recharts
- **Excel Parsing**: SheetJS (xlsx)
- **PDF Export**: jsPDF

## Getting Started

### Prerequisites

- Node.js 18.17.0 or later

### Installation

1. Clone the repository
```bash
git clone https://github.com/yourusername/retirement-planning.git
cd retirement-planning
```

2. Install dependencies
```bash
npm install
```

3. Run the development server
```bash
npm run dev
```

4. Open [http://localhost:3000](http://localhost:3000) in your browser

## Usage

1. Navigate between calculators using the tabs at the top of the page
2. Adjust parameters using sliders and input fields
3. View real-time results in charts and tables
4. Export results to CSV for further analysis

## Deployment

This application can be easily deployed to Vercel, Netlify, or AWS S3.

```bash
npm run build
```
